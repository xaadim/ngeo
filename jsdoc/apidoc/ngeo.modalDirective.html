Class: modalDirective

modalDirective

new modalDirective($parse) → {angular.Directive}

Provides the "ngeoModal" directive. This directive allows displaying a modal window. The directive takes care or the show/hide actions.

This directive requires Bootstrap's modal classes and associated jQuery plugin.
