ASITVD source example

GeoMapFish Authentication example

This example shows how to use the gmf-authentication directive to insert an authentication panel in a GeoMapFish page.

GeoMapFish Layer Tree Example

This example shows how to use the gmf.Layertree directive. This layertree needs a source object (that can change) that describes layers like the c2cgeoportal themes service. The wmsUrl is used by internals WMS layers but you can use WMS externals layers and WMTS layers too.

GeoMapFish's Mobile Background Layer Selector example

This example shows how to use the gmf-mobile-background-layer-selector directive to select the map background layer.

Mobile Measure Example

This example shows how to use the gmf-mobile-measure-length and gmf-mobile-measure-point directives in combination with ngeo.interaction.MobileDraw interaction objects to allow mobile users to measure distances and know the center coordinate on their devices. The point measure also uses the gmf-altitude service to also obtain the height of the location as well.

GeoMapFish Mobile Query example

This example shows how to use the ngeo-mobile-query directive in combination with the gmf.QueryManager. The QueryManager fetches the themes returned by the theme service and adds one source to the query service per layer definition found. In this example, the layer tree is responsible of creating the layer. You can switch theme and turn on/off layers to see the impact it has on results returned by the query service.

GeoMapFish Permalink example

This example demonstrates the use of the ngeo-permalink service.

Search GeoMapFish example

This example shows how to use the gmf-search directive, which is based on Twitter's typeahead component.

Simple GeoMapFish example

This example shows how to use the gmf-map directive to insert an OpenLayers map in a GeoMapFish page.

GeoMapFish Theme Selector Example

This example shows how to use the gmf.themeselector directive from a c2c geoportal theme service.
